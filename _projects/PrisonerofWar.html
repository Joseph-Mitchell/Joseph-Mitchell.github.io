Summary

This project was created as a part of a module from my college Games Development course which was intended to help us showcase our ability to work within a team. Apart from putting forward initial proposals as well as helping to create planning documentation, my main role within the team was to create artwork for use outside of the main game, mostly consisting of UI art.
